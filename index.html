Let's Have Some Order! ... In A 2D Array

2D arrays are represented in terms of rows and columns. 2D arrays can be read in different ways, like across rows, up or down columns. Row-major order refers to the ordering of elements in a 2D array, where traversal occurs across each row. Column-major order refers to traversal down each column. There are performance differences between row-major order and column-major order, related to how the 2D array is organized, but these differences are outside the scope of this course.
